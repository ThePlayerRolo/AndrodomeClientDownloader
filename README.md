# Androdome Client Downloader
A python script to make downloading old roblox clients from Androdome easier.

NOTE: This fully works with repackedged and fully found clients from 2008-2015, anything later and/or imcomplete will probably not work

# Install Prerequsites
1. Install the latest version of Python at https://www.python.org/
2. Run InstallDepedences.bat
 
## Usage
Run the following command:
```AndrodomeClientDownloader.py -v versionhash```
Where versionhash is the hash of your client
