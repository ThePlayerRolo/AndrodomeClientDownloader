# Androdome Client Downloader
A python script to make downloading old roblox clients from Androdome easier.

NOTE: This fully works with repackaged and fully found clients from 2008-2014, anything later and/or incomplete may not work

Please report any issues!

# Install Prerequsites
1. Install the latest version of Python at https://www.python.org/
2. Run InstallDepedences.bat or ```python -m pip install wget python -m pip install requests ```
 
## Usage
Run the following command: <br>
```AndrodomeClientDownloader.py -v versionhash``` <br>
Where versionhash is the hash of your client
